Спасибо за замечания и рекомендации!
Теперь стало более понятно, что требуется в этой работе.

В соответствии с твойей рекомендацией обозначить, что находится в контуре фермы, а что в головной компании, изменил варианты решений.
Первоначально предполагал, что решение для MVP касается только автоматизации фермы: центрального сервера и его агентов. Головная компания оставалась не прописанной в решении и предполагалась за будущими API для мобильной/веб-платформы и выгрузки метрик.

В текущей версии ввел головную компанию. Центральный сервер фермы будет отвечать за передачу данных к головной офис. Рабочее место менеджера, не связанное с оперативной реакцией на инцеденты, аналитика, хранение данных ушли в головной офис.

Как альтернативный вариант прописал решение с подключением всех рабочих мест серверу головной компании с очевидными задержками в управлении устройствами/реакцией на инцеденты.

В папке с каждым заданием добавил файл readme.txt с подробными описаниями изменений в версии 1.1.
